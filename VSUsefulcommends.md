# Useful commands for C/C++ programming in VS Code

Hotkey list:

| OPTION               | Shortcut             |
| -------------------- | -------------------- |
| Open Command Pallete | ctrl+shift+p         |
| Go to previous line  | ctrl+shift+num_pad_- |
| Go to next line      | ctrl+shift+num_pad_+ |
| Debug                | ctrl+f5              |
| Build                | Ctrl+Shift+B         |
| Caps Lock            | ⇪                    |
| Tab                  | ⇥                    |
| Esc                  | ⎋                    |
| Power                | ⌽                    |
| Return               | ↩                    |
| Delete               | ⌫                    |
| Up                   | ↑                    |
| Down                 | ↓                    |
| Left                 | ←                    |
| Right                | →                    |
